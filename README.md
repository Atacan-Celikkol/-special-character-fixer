# Special Character Fixer
Fixes the language specific special characters in text files. Especially for the <b>*.srt*</b> or <b>*.txt*</b> files.
<br>
<br>
**For example:**
```diff
- Iþýðýn Tanrýsý'nýn öyle istemesi.
+ Işığın Tanrısı'nın öyle istemesi
```
